# Código a tres dimensiones (TAC) en un EDTS

La gramática de C (mini-C para esta tarea) se implementará en un EDTS con código de tres direcciones para el árbol.
Código de tres direcciones: Es la traducción semántica del árbol a instrucciones simples. Es una representación intermedia del código, que los compiladores generan ANTES de generar ensamblador.
